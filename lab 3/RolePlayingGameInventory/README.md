# Лабораторная 3: RPG Inventory

Система инвентаря в стиле RPG с оружием, броней, зельями, квестовыми предметами, фабриками и сервисами экипировки игрока.

## Основные идеи

- Интерфейсы для оружия, брони, зелий, экипируемых и улучшаемых предметов.
- Фабрики для создания разных наборов предметов.
- Правила веса инвентаря и квестовые предметы с нулевым весом.
- Сервис экипировки защищает от повторного применения эффектов и сбрасывает состояние при снятии предмета.
- xUnit-тесты для фабрик, веса инвентаря, прокачки предметов, экипировки и зелий.

## Запуск

```bash
dotnet run --project "RolePlayingGameInventory/RolePlayingGameInventory.csproj"
dotnet test "RolePlayingGameInventory.sln"
```
