# Лабораторная 4: Order System

Система заказов ресторана, демонстрирующая несколько классических ООП-паттернов в небольшой предметной области.

## Основные идеи

- Strategy для обычной, VIP- и быстрой доставки.
- Factory Method для создания заказов с нужной стратегией расчета стоимости.
- Decorator для модификации блюд.
- State для переходов между статусами заказа.
- Observer для уведомлений об изменении статуса заказа.
- xUnit-тесты для стратегий расчета цены, декораторов, переходов состояния и обработки отсутствующих заказов.

## Запуск

```bash
dotnet run --project "OrderSystem/OrderSystem.csproj"
dotnet test "OrderSystem.sln"
```
