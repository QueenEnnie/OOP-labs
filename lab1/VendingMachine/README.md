# Лабораторная 1: Vending Machine

Консольная симуляция вендингового автомата с доменной моделью для товаров, остатков, выручки и действий администратора.

## Основные идеи

- Инкапсуляция количества товара и выручки автомата.
- Проверка покупки по наличию товара и внесенной сумме.
- Отделение тестируемой бизнес-логики от консольного взаимодействия.
- xUnit-тесты для продажи, расчета сдачи, сбора выручки и пополнения товаров.

## Запуск

```bash
dotnet run --project "VendingMachine/VendingMachine.csproj"
dotnet test "VendingMachine.sln"
```
