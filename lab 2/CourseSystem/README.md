# Лабораторная 2: Course System

Консольное приложение для управления курсами: создание курсов, добавление преподавателей, назначение преподавателей на курсы и просмотр студентов.

## Основные идеи

- Интерфейсы `ICourse` и `IPerson` для общих контрактов.
- Реализации онлайн- и оффлайн-курсов.
- `ManagementSystem` как основной доменный сервис.
- Валидация на уровне бизнес-логики: дублирующиеся ID, некорректные типы курсов, отсутствующие сущности.
- xUnit-тесты для курсов, преподавателей, назначений и ошибок валидации.

## Запуск

```bash
dotnet run --project "CourseSystem/CourseSystem.csproj"
dotnet test "CourseSystem.sln"
```
